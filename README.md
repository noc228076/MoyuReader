# Moyu Reader (摸鱼阅读器) 🐟

一款专为打工人设计的**隐蔽式桌面沉浸阅读器**。你可以用它在任何工作界面（甚至在写代码、查文档时）隐蔽地看小说。完全无边框、背景透明，可以高度伪装成报错信息、代码注释或者文档记录。

![Python Version](https://img.shields.io/badge/python-3.10+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 🌟 核心特性

- **👻 极致伪装**：无边框设计，支持任意调节透明度、字体颜色和每次显示的行数。
- **🚀 零宽操作**：鼠标停留在文字上往下滚立刻翻页，右下角自由拖拉调整大小。
- **🚨 终极安全**：
  - **左键双击**：瞬间原地隐身。
  - **全局老板键**：支持全局按键（默认 `Alt + Shift + H`，可在软件内随时重新录制热键）彻底隐藏或唤醒窗口，即便你在其他应用内也能生效。
- **📚 多书记忆**：自动记录看过的最近 10 本小说，并且**每本书的进度独立保存**，完全离线运行。
- **📝 格式支持**：支持市面上最广泛的 **`TXT`** 纯文本以及主流电子书 **`EPUB`** 格式智能提取。

## 🎮 快速使用体验
针对没有编程经验的用户，您可以直接在顶部 [Releases] 中下载打包好的绿色单文件版 `.exe`。

### 基础操作指南
1. **启动应用**：双击运行 `moyu_reader.exe`。
2. **载入书籍**：在出现的文字上点击 **鼠标右键** -> `📂 载入 TXT/EPUB`。
3. **调整位置与大小**：
   - 鼠标按住文字区域即可**拖动窗口位置**。
   - 鼠标悬停在文字**右下角**边缘，拖拉即可调整窗口高度和宽度。
4. **摸鱼翻页**：光标移动到软件区域上，**向下滚动鼠标滚轮**即可翻下一页，向上翻上一页。
5. **设置快捷隐藏键**：右键菜单 -> `⚙️ 自定义老板键 (一键隐藏)`。在弹出的窗口中直接按下您想设定的键盘组合键即可完成绑定！

## 💻 源码运行与自行编译

### 1. 准备环境
确保您的电脑上安装了 Python（建议 3.8 及以上）。
克隆本项目至本地后，安装所需的依赖：

```bash
pip install -r requirements.txt
```
*(如果需要使用老板键功能，请确保拥有足够的执行权限，或在虚拟环境下安装)*

### 2. 本地直接运行
```bash
python moyu_reader.py
```

### 3. 打包免安装 Executable (.exe)
如果你想把软件分享给公司的同事或者放在 U 盘里带走，可以用 `pyinstaller` 进行单文件打包：
```bash
pyinstaller --noconsole --onefile moyu_reader.py
```
打包成功后，单文件程序将生成在自动创建的 `dist/` 目录下。
